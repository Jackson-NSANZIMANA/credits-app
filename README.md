# Credits App

A credits management application for small businesses.

## Tech Stack

- React.js (frontend)
- Node.js + Express (backend)
- TailwindCSS (UI)
- JWT authentication

## How to Run

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```
